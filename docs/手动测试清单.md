## 测试前准备

### 确认服务状态

在终端执行以下命令确认服务运行正常：

```bash
# 检查 OpenClaw Gateway
curl -s http://localhost:18789/health

# 检查 Claude Code Proxy
curl -s http://localhost:8082/health

# 检查数据库连接
psql -d ai_cs_support -c "SELECT COUNT(*) FROM customer"
```

**预期结果：**
- Gateway 返回 `{"ok":true,"status":"live"}`
- Proxy 返回 `{"status":"healthy"}`
- 数据库返回客户数量（应为3条）

### 打开飞书

1. 打开飞书应用
2. 找到 **AI客服** 机器人
3. 开始测试

## 注意事项

1. **上传文件格式**：支持 .xlsx、.xls、.pdf
2. **置信度说明**：
   - 绿色（≥90%）：高置信度，可直接使用
   - 橙色（70-89%）：中置信度，建议核对
   - 红色（<70%）：低置信度，需要人工确认
3. **测试环境**：使用真实数据进行测试
4. **问题反馈**：如有异常，记录具体错误信息
