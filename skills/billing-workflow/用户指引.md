## 关键公式

### 标准件数量计算
```
标准件数量 = 计划出库数量 / 换算率 × 计费系数
```

### 运输费计算
```
运输费 = max(标准件数量 × 单价, 最低收费)
```

### 计费系数计算
```
固定SKU：合同约定系数 = 1.0
非固定SKU：
  W≥30kg 或 V>0.1m³ → ceil(V/0.05)
  25kg≤W<30kg 或 0.08m³<V≤0.1m³ → 2.0
  15kg≤W<25kg 或 0.05m³<V≤0.08m³ → 1.2
  W<15kg 且 V≤0.05m³ → 1.0
```

执行账单审核时，必须检查：
- [ ] 基础配置数据是否完整？
