## 📌 华鼎模板字段批注规则（2026-05-12 14:04更新）

### 字段必填规则

| 字段 | 必填性 | 规则说明 |
|------|--------|----------|
| 序号 | ✅必填 | 同单一组商品序号相同，多单导入时序号必须连续（1,2,3,4...不能跳号如1,2,4,5） |
| 门店编号/门店三方编码 | ⚠️二选一 | 至少填一个 |
| 仓库编码 | ✅必填 | 数字，如5 |
| 加急程度 | ✅必填 | 0=普通，1=加急 |
| 商品SKU编号/商品三方SPEC编号 | ⚠️二选一 | 至少填一个 |
| 单位类型 | ✅必填 | **小单位/大单位**（不是0/1） |
| 出库数量 | ✅必填 | 数字 |
| 指定库存状态 | ✅必填 | 默认"正常" |
| 出库类型 | ✅必填 | 201=销售订单，202=采购退回，203=其他 |
| 配送方式 | ✅必填 | 专配/快递自提/市场车/快递代发 |
| 指定车型（专配） | ⚠️条件 | 配送方式=专配时必填 |
| 是否垫付 | ⚠️条件 | 配送方式=市场车/快递代发时必填 |
| 付款方式 | ⚠️条件 | 配送方式=市场车/快递代发时必填 |
| 快递公司 | ⚠️条件 | 业务类型=B2C时必填 |
| 单价/总金额 | ❌选填 | 不填 |
| 是否制定批次 | ✅必填 | 0=否，1=是（默认填"否"） |
| 批次号 | ❌选填 | 不填 |
| 生产日期 | ❌选填 | 格式YYYY-MM-DD，不填 |
| 备注 | ❌选填 | 从订单规格提取（如"2kg*10袋"） |
| 生产厂家编号 | ❌选填 | 不填 |
| 门店收货地址编码 | ❌选填 | 收货方为门店时留空用默认地址 |
| 三方单号 | ✅必填 | 订单编号 |
| 业务模式 | ✅必填 | B2C/B2B |
| 业务类型 | ❌选填 | 默认B2B；B2C时联系电话+收货地址+C端快递公司必填 |
| 收货人 | ✅必填 | 联系人 |
| 联系电话 | ⚠️条件 | B2C时必填 |
| 收货地址 | ⚠️条件 | B2C时必填 |
| C端快递公司 | ⚠️条件 | B2C时必填 |

### 单位类型规则（重要）
- 0 = 小单位
- 1 = 大单位
- 实际填写：**小单位/大单位**（文字）

### 联动规则
```
配送方式 = "专配" → 指定车型 必填
配送方式 = "市场车" 或 "快递代发" → 是否垫付 + 付款方式 必填
业务类型 = "B2C" → 联系电话 + 收货地址 + C端快递公司 必填
```
