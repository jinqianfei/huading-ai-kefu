## Skill 1: skill_order_to_huading_template

### 功能
将客户订单Excel转换为华鼎系统可导入的出库单模板（31字段）

### 输入
- 客户订单Excel文件
- 数据库连接 (db_config)

### 输出
- 华鼎出库单模板Excel文件

### 处理流程
```
Step 1: 订单解析 → 提取订单号、门店名、商品明细
Step 2: 门店匹配 → 从store_list表查门店编号、仓库、收货人
Step 3: 获取owner_code（货主ID）→ 询问用户确认货主
Step 4: SKU匹配 → 从shipper_sku_mapping表查系统SKU编号（用owner_code过滤）
Step 5: 生成模板 → 填充31字段，输出Excel
```

### 货主匹配逻辑

**匹配方式：** 通过门店名 → 查store_list → 获取owner_code（货主ID）

**货主-品牌对照：**
| 货主公司全称 | 品牌名称 | 货主ID (owner_code/shipper_id) |
|--------------|---------|-------------------------------|
| 河南上黎供应链管理有限公司 | 制茶青年 | HZ2023061500002 |
| 天津王口镇店 | - | CUSTOMER-WANGKOU |
| 长沙广承供应链有限公司 | - | CUSTOMER-GUANGCHENG |

**确认流程：**
```
1. 解析订单 → 得到门店名
2. 查store_list → 获取owner_code + 门店信息
3. 展示货主匹配结果 → 询问用户确认
4. 用户确认后 → 用owner_code过滤SKU匹配
5. 匹配失败时 → 提示用户检查SKU映射
```

### 字段映射

**必须匹配的字段（无默认值）：**
| 字段 | 数据来源 |
|------|---------|
| 门店编号 | store_list表 |
| 仓库编码 | store_list表 → warehouse_code_mapping |
| 商品SKU编号 | shipper_sku_mapping表（用owner_code过滤） |
| 收货人/电话/地址 | store_list表 |
| 货主ID (owner_code) | store_list表（从门店匹配获取） |

**有默认值的字段：**
| 字段 | 默认值 |
|------|--------|
| 加急程度 | 0（普通） |
| 指定库存状态 | 正常 |
| 出库类型 | 201（销售订单） |
| 配送方式 | 共配 |
| 是否垫付 | 否 |

**固定为空的字段：**
- 单价、总金额、批次号、生产日期、三方编码...

## Skill 2: billing-workflow

### 功能
AI账单制作全流程

### 三步流程
```
步骤1: 合同规则提取（用户触发"更新合同"）
  → 输入：合同（图片/PDF）
  → 输出：规则库更新

步骤2: SKU系数计算（用户触发"更新商品系数"）
  → 输入：商品信息Excel + 规则库
  → 输出：每个SKU的计费系数

步骤3: 费用计算（用户触发"计算账单"）
  → 输入：出库明细 + 规则库 + SKU系数
  → 输出：月度账单
```

### 规则库结构
```
knowledge/rules/
└── {customer_id}/
    ├── contract.json      # 合同规则
    ├── coefficients.json  # SKU计费系数
    └── latest.json        # 最新版本汇总
```

### 计费公式
```
标准件数量 = 计划出库数量 / 换算率 × 计费系数
运输费 = max(标准件数量 × 单价, 最低收费)
```

### 标准件系数规则
```
固定SKU（合同约定）→ 系数 = 1.0
非固定SKU：
  tier3: >30kg 或 >0.1m³ → max(重量/15kg, 体积/0.05m³)
  tier2: 25-30kg 或 0.08-0.1m³ → 2.0
  tier1: 15-25kg 或 0.05-0.08m³ → 1.2
  标准件: ≤15kg 且 ≤0.05m³ → 1.0
```

## 约束

1. **多货主支持** - 不默认货主，通过门店名匹配后必须用户确认
2. **规则必须确认** - 合同解析后必须用户确认才能用于账单
3. **阈值严格** - 账单差异>0.01元必须告警
4. **映射置信度** - SKU匹配<80%需人工确认
5. **数据格式** - 仅支持华鼎系统导出的标准Excel
